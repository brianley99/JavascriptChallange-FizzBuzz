FizzBuzz App

When it counts you FizzBuzz. A Javascript coding exersise using loops. Write a program that prints the numbers from 1 to 100 and for multiples of '3' display 'Fizz' instead of the number and for the multiples of '5' display 'Buzz'. If a multiple of both display 'FizzBuzz'.

Instructions

Eneter a fizz and a buzz value. Press the button and the values will appear.
